# Rule Template

This template shows the expected format for rules in this skill.

## Frontmatter Format

```yaml
---
title: "Rule Title"
impact: CRITICAL | HIGH | MEDIUM | LOW
tags:
  - tag1
  - tag2
---
```

## Content Structure

1. **Overview** - What this rule covers and why it matters
2. **Core Concepts** - Principles and mental models
3. **Reference Tables** - Attribute tables, mapping tables, namespace tables
4. **Common Mistakes** - Anti-patterns with corrections
5. **References** - Links to authoritative sources

## Writing Guidelines

- Use concrete examples over abstract explanations
- Include both "good" and "bad" patterns where applicable
- Keep examples copy-pasteable
- Link to related rules when relevant
