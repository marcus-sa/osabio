# Rule Template

This template shows the expected format for rules in this skill.

## Frontmatter Format

```yaml
---
title: "Rule Title"
impact: CRITICAL | HIGH | MEDIUM | LOW
tags:
  - tag1
  - tag2
---
```

## Content Structure

1. **Official Documentation** - Links to authoritative sources
2. **Core Concepts** - Key principles and mental models
3. **Implementation Details** - Code examples and configuration
4. **Best Practices** - Do's and don'ts
5. **Troubleshooting** - Common issues and solutions

## Writing Guidelines

- Use concrete code examples over abstract explanations
- Include both "good" and "bad" patterns where applicable
- Keep examples copy-pasteable
- Link to related rules when relevant
